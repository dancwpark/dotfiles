# NERDTree
## Bring up NERDTree
* `:NERDTreeToggle`
* Set up to a mapping later...
  * `map <C-n> :NERDTreeToggle<CR>a
  * CTRL+n => open up NERDTree
## Create a new file or directory
* Bring up NERDTree
* `m` to bring up menu
* `a` to add child node
  * add `\` to end for directory

# VIm/nvim
## Open up file in new tab
* `:tabf $file`
## Tab navigation
* `gt` - next tab
* `gT` - previous tab
* `#gt` - #th tab
* `:tabn`
* `:tabp`
